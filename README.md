# О проекте

Проект разработан в рамках хакатона "СпортХак"



# Вид JSON-файла :
> В корне проекта, нужно создать папку local и в нее закинуть config.json 
> в формате:
```
{
    "secret_key": "sectet_key",
    "db_type": "sqlite3",
    "database": "db_name",
    "user": "db_user",
    "host": "localhost",
    "password": "db_password",
    "port": "1234"
}

```
